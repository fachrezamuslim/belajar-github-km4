“Javascript & DOM #2 - DOM Tree”

DOM Tree atau pohon DOM adalah representasi elemen-elemen html dalam browser.

Seluruh elemen html disebut sebagai simpul atau node.

Tiap simpul atau node mempunyai tipe: semua elemen html bertipe element (head, body, h1,h2,h3,h4,h5,p) tags node adalah simpul yang berisi text elemen yang mempunyai atribut misal (href) disebut attribute dan masih banyak lagi

NodeList VS HTMLCollection keduanya merupakan kumpulan node struktur datanya mirip array nodelist dapat berisi node apapun, sedangkan HTMLCollection harus berisi elemen HTML HTMLCollection bersifat live sedangkan NodeList tidak.

Struktur hirarki DOM Tree

Root Node node yang menjadi sumber dari semua node lain di dalam DOM document Parent Node node yang berada 1 tingkat diatas node yang lain body adalah parent dari h1, p,p, div dan a Child Node node yang berada di 1 tingkat dibawah node yang lain h1 adalah child dari body

TUTORIAL JAVASCRIPT DOM (BAHASA INDONESIA)

Tipe data DOM Document : Representasi dari halaman web Node : base class untuk semua tipe data di DOM, seperti Document, Element dan Attr. Element: representasi object element di halaman web, misal head, body, form, input dan lain-lain. NodeList : Array/collection yang berisikan Node. Attr : atau Attribute, merupakan representasi object attribute dalam element. NamedNodeMap : collection yang berisikan Attr dalam bentuk Map.

Document object tidak perlu dibuat secara manual karena secara default sudah dibuat oleh browser saat halaman web dimuat.

Document memiliki banyak method, method yang terdapat di document digunakan untuk memanipulasi data DOM, misal membuat Node, Element, Attribute atau mengambil dan menyeleksi Node di dalam document. contoh getElementById

Node adalah base class dari Document, Element dan Attr. Semua fitur yang dimiliki di Node dimiliki diturunannya. Node dalam DOM bentuknya adalah Tree (pohon), artinya dengan Node, kita bisa melihat Parent (node di atasnya), Children (node dibawahnya), Sibling (node di sebelahnya)

Element merupakan Node yang berbentuk element, biasanya adalah element HTML, misal

html head body div table

Selain mengambil element menggunakan id, bisa juga membuat object element baru, dan menambahkannya ke halaman web, dengan menggunakan method createElement(tag) pada Document.

NodeList adalah kumpulan dari Node. Biasanya NodeList digunakan ketika kita menyeleksi banyak Node sekaligus, misal ketika kita ingin mengambil semua children di Node.

Live vs Static NodeList NodeList memiliki 2 tipe, yaitu Live dan Static Live artinya perubahan yang terjadi pada NodeList nya, akan merubah semua NodeList yang sama Static artinya perubahan yang terjadi pada NodeList nya, tidak akan merubah semua NodeList yang sama Saat menggunakan element.childNodes, NodeList tersebut bersifat Live Saat menggunakan document.querySelectorAll(), NodeList tersebut bersifat Static

Attr Value : Untuk mendapatkan value secara langsung, bisa menggunakan method getAttribute(name) di Element.

Untuk membuat Attr, bisa menggunakan document.createAttribute(name) Lalu untuk menambahkannya ke Element, bisa gunakan element.setAttributeNode(attr) Atau juga bisa langsung membuat Attr langsung dengan name dan value nya menggunakan element.setAttribute(name, value).

NamedNodeMap merupakan kumpulan dari Attr. Mirip seperti NodeList, namun isinya adalah Attr.

Text Node : Saat membuat HTML, kadang kita membuat teks tanpa tag saat halaman web tersebut direpresentasikan dalam DOM, text tanpa tag bukanlah merupakan Element, melainkan Node, Lebih tepatnya adalah Text Node.

Text Node bisa dibuat dengan menggunakan method document.createTextNode(string) Hasilnya berupa Text yang bisa di tambahkan ke Node lain.

Event handler : Node memiliki kemampuan bereaksi terhadap suatu kejadian, misal kejadian di klik, mouse di atas node tersebut, dan lain-lain. Ada dua cara kita menambahkan Event Handler ke Node, menggunakan Event Target atau Global Event Handler.

Event target : Event Target merupakan parent class dari Node, artinya semua Node memiliki kemampuan dari Event Target. Pada Event Target, bisa menggunakan method addEventListener(type, callback) untuk menambahkan Event Handler.

Global Event Handler bukanlah sebuah super class, hanya kontrak yang bisa kita gunakan untuk menambah event handler sesuai dengan type event nya

DOM memiliki banyak jenis event, tidak hanya click. Jenis event yang bisa digunakan biasanya disesuaikan dengan target event nya, misal pada element video, ada event play, pause, sedangkan pada element button, tidak ada event play atau pause.

DOM style : DOM juga bisa digunakan untuk memanipulasi attribute style pada element dengan lebih mudah. Bisa langsung gunakan element.style.name = value.

Saat mengubah sebuah style, penamaan style menjadi berubah Pada style css, rata-rata menggunakan format lowercase dan -, misal background-color Sedangkan pada DOM Style, menggunakan format pascalCase, misal backgroundColor

Selain textContent, terdapat property lain bernama innerText untuk mendapatkan isi text sebuah element.

Jika Text Content dan Inner Text hanya mengambil text saja, Inner HTML akan mengambil seluruh element HTML nya sebagai text.

Window:

Window merupakan representasi window yang berisikan DOM Document Beberapa JavaScript function berasal dari object Window, seperti alert(), confirm() dan prompt(). Ada banyak property, method dan event yang bisa digunakan pada window Untuk menggunakan Window, tidak perlu membuat objectnya, cukup gunakan kata kunci window.

Query Selector

Query Selector merupakan method untuk menseleksi node sesuai dengan pola yang ingin cari. Terdapat dua method untuk Query Selector : Document.querySelector(pola) digunakan menyeleksi node pertama yang sesuai dengan pola Document.querySelectorAll(pola) digunakan untuk menyeleksi semua node yang sesuai dengan pola

Pola untuk Query Selector adalah menggunakan CSS Selector. Penggunakan Pola nya mirip ketika membuat Selector di CSS.

Universal Selector merupakan selector untuk menyeleksi semua element dengan menggunakan karakter *

Type Selector adalah selector yang digunakan untuk menyeleksi tag type HTML yang dipilih. Untuk menggunakannya, bisa langsung sebutkan nama tag nya.
