Javascript & DOM #1 - Pendahuluan

DOM

Untuk membuat halaman web menjadi interaktif dan menarik dapat menggunakan Document Object Model. DOM adalah antar muka pemrograman untuk HTML yang merepresentasikan halaman web, sehingga program dapat mengubah dan memanipulasi struktur nya. Atau DOM bisa juga di artikan representasi dari elemen elemen HTML pada dokumen yang menjadi sebuah object. DOM yang tersimpan tadi akan membentuk pohon hirarki DOM atau DOM TREE. Komponen dari DOM adalah Elemen HTML, atribut, text dan lain lain

DOM TREE

DOM TREE adalah hirarki yang tersimpan pada dokumen. Dimana DOM tree yang terbetuk akan sesuai dengan hirarki HTML yang di tulis. DOM TREE berguna ketika ingin menelusuri sebuah elemen yang akan digunakan. Dari setiap komponen dom di sebut juga dengan Node.


Javascript & DOM #2 - DOM Tree

DOM TREE

DOM TREE adala hirarki berbentuk pohon yang merepesentasikan elemen, elemen dapat berisikan elemen html ,atribut, text dan lain lain yang tedapat pada browser. Setiap elemen dianggap sebagai simpul. Node pada DOM TREE memiliki banyak tipe, seperti tag html memiliki tipe element dan lain lain.

NodeList vs HTML Collection

NodeList adalah pemilihan lebih dari satu node dan memiliki elemen bebas, sedangkan HTML Collection berisikan node dengan hanya elemen HTML dan lebih dari satu. Dari perbedaan ini menunjukan cara yang berbeda ketika mengelola nya. NodeList dan HTML Collection sama sama kumpulan node. Dimana struktur data nya mirip array. HTMLCollection akan bersifat live sedangkan NodeList tidak

Struktur Hierarki DOM Tree

Konsep dari Struktur Hierarki DOM Tree mirip dengan konsep pohon keluarga dimana memiliki parent dan juga child. Struktur Hierarki DOM di awali dengan ROOT Node yang akan menjadi sumber dari semua NODE lain di dalam dom. Parent Node adalah node yang berada 1 tingkat dari node lain. Child Node adalah node yang berada 1 tingkat dari node lain.

DOM

Ketika sebuah website di load oleh browser, maka browser akan membuat Document Object model untuk halaman website nya. DOM ini berbentuk tree/pohon dimana setiap cabang nya berisi obhcet dari jenis elemen html. JavaScript dapat memanipulasi DOM, artinya dengan JavaScript, kita dapat mengkases tiap object pada DOM hingga dapat memanipulasi nya. DOM bukanlah sebuah fitur atau bahasa pemrograman, melain DOM hanya repesentasi dari Oject dari struktur halaman web. DOM adalah bagian dari Web api dan bukan bagian dari JavaSCript. DOM dapat digunakan oleh bahasa pemrograman lain tidak hanya javascript.

Tipe Data

DOM sangat sederhana dan mempunyai manfaat dalam penggunaan nya. Maka dari itu DOM memiliki representasi data namun tidak terlalu banyak. Berikut adalah Tipe data dalam DOM :

  • Document: repesentasi dari halaman web
  • Node: base class dari setiap data di DOM, seperti Document, element, attribute, text dan lain lain
  • Element: representasi dari object elemn di halaman web seperti head, body, form, input dan lain lain
  • Node List: array/collection/kumpulan yang berisikan node
  • Attr: atribut, representasi dari object atribut pada element
  • NameNodeMap: Collection berisikan Attr dalam bentuk map

Document

Document adalah repesentasi dari halaman web yang terbentuk secara otomatis di browser. Untuk meakses Document dapat menggunakan object document. Document memiliki banyak property yang berfungsi untuk melihat semua data yang tedapat dalam document halaman web. Contoh document property adalah, title, body, head, images, contentType, dan masih banyak lagi. Document juga memiliki Method yang banyak. Method digunakan untuk memanipulasi data DOM, misalkan untuk membuat Node, Element, Attribute, atau mengambil dan menyeleksi Node dalam document.

Node

Node adalah base class dari Document, Element dan Attr. Artinya semua fitur dari Node akan diturunkan ke node berikutnya. Node berbentuk Tree, yang artinya Node mempunyai Parent (Node di atas nya), Child(Node di sebelah nya), SIbling (Node di sebelah nya). Contoh method untuk mengambil node dapat menggunakan getElementById. Node juga memiliki banyak property, property node memiliki informasi reference/ refrensi ke node lain nya seperti node parent atau node children dan node sibling nya. Node juga memiliki Method, Method dalam node digunakan untuk memanipulasi data dalam Node, misalkan ingin menambah/menghapus node children/sibling.

Element

Element adalah Node yang berbentuk element, biasanya berisikan element HTML seperti , , dan lain lain. Elemetn memiliki property, karena element adalah turunan dari Node maka property pada node dapat digunakan untuk element dan ada juga tambahan nya. Element juga memiliki banyak method, karena element adalah turunan dari Node maka method pada node dapat digunakan untuk element dan ada juga tambahan method seperti evenListener dan lain lain. Dapat juga membuat element baru dengan method dari document.
