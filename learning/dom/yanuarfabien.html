Video 1: Javascript & DOM #1 - Pendahuluan

Document Object Model atau disingkat DOM dapat dipecah menjadi 3 kata yaitu Document, Object, dan Model. Ketiga kata ini jika didefinisikan masing masing menjadi:

  1. Document: Isi dari sebuah website
  2. Object: Sebuah nilai atau tipe data dalam JavaScript yang memiliki property dan method yang strukturnya dapat dimanipulasi
  3. Model: Representasi terhadap sesuatu

Sehingga dari ketiga kata tersebut, definisi sederhana dari DOM adalah representasi elemen HTML pada dokumen yang nantinya akan menjadi sebuah object yang strukturnya dapat dimanipulasi. Pada memory, DOM akan tersimpan sebagai pohon hirarki DOM atau biasa disebut DOM Tree.

Ada beberapa hal yang dapat dipelajari untuk menguasai DOM yaitu:

  1. DOM Selection: Bagaimana cara untuk memilih atau mengakses elemen HTML dengan JavaScript
  2. DOM Manipulation: Bagaimana cara untuk memanipulasi elemen HTML dengan JavaScript
  3. DOM Traversal: Bagaimana cara untuk menelusuri elemen HTML lewat HTML lain
  4. Event Handling: Bagaimana cara melakukan sesuatu saat ada trigger pada DOM

Video 2: Javascript & DOM #2 - DOM Tree

DOM Tree adalah representasi elemen-elemen HTML pada browser. Tiap elemen pada DOM Tree dapat disebut dengan node atau simpul. Simpul paling atas disebut simpul root atau document. Node dari DOM Tree memiliki banyak tipe, namun dalam mempelajari DOM Tree biasanya hanya berfokus pada 3 tipe yaitu:

  1. Elemen
  2. Text
  3. Document

Ada 2 istilah tipe dari kumpulan node yang harus dipahami yaitu:

  1. NodeList: Kumpulan node yang dipilih (lebih dari 1 node) apapun tipenya
  2. HTMLCollection: Kumpulan node yang dipilih (lebih dari 1 node) dengan tipe elemen

Struktur data dari kumpulan node mirip seperti array. Adapun struktur hirarki dari DOM yaitu:

  1. Root Node: node yang menjadi sumber dari semua node lain di dalam DOM
  2. Parent Node: node yang berada 1 tingkat diatas node lain
  3. Child Node: node yang berada 1 tingkat dibawah node lain
  4. Siblings Node: node yang memiliki parent yang sama

Video 3: TUTORIAL JAVASCRIPT DOM (BAHASA INDONESIA)

Ketika sebuah website di load oleh Browser, maka Browser akan membuat Document Object Model untuk halaman website tersebut dalam bentuk pohon atau tree yang dapat dimanipulasi dengan JavaScript. Adapun beberapa tipe data dalam DOM yaitu:

  1. Document: Representasi dari halaman web. Dalam satu halaman web terdapat satu document. Document memiliki property yang dapat digunakan untuk melihat secara detail data yang terdapat pada document dari suatu halaman web. Selain itu, document juga memiliki method yang biasanya dapat digunakan untuk memanipulasi DOM didalamnya
  2. Node: Base class untuk semua tipe data di DOM, seperti Document, Element, dan Attr. Node dalam DOM bentuknya adalah Tree (pohon). Artinya, dengan node dapat melihat Parent (node diatas nya), Children (node dibawahnya), Sibling (node disebelahnya). Node memiliki property yang berisi informasi yang mereferensi ke node lainnya seperti ke node parent-nya, node children-nya ataupun node sibling/sebelahnya. Selain itu, node juga memiliki method untuk memanipulasi data yang terdapat pada node tersebut
  3. Element: Representasi object element HTML di halaman web, misal head, body, form, input, dan lain-lain. Element merupakan turunan dari node yang berarti semua property dari node dapat digunakan di element. Selain itu, karena element turunan dari node, element juga dapat menggunakan method yang dimiliki oleh node
  4. NodeList: Array/collection yang berisikan node
  5. Attr/Attribute: Representasi object attribute dalam element
  6. NamedNodeMap: Collection yang berisikan Attr dalam bentuk Map
