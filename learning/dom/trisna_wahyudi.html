Javascript & DOM #1 - Pendahuluan

DOM (document object model) merupakan programing interface untuk HTML yang merepresentasikan halaman web sehingga program dapat mengubah dan memanipulasi strukturnya. ~ Mozila

Ketika kita membuat sebuah file html, yang kemudian dibuka di browser, maka element-eement dalam html tersebut akan dibaca dan direpresentasikan sebagai object oleh browser. Representasi ini disimpan dengan nama pohon hirarki DOM (DOM Tree).

Document sebenarnya meurpakan node dari object yang lebih tinggi, yaitu Window. didalam window terdapat beberapa node yaitu: document, history, dan location.

Dengan menggunakan dom, kita dapat mengakses dan manipulasi elemen-elemen html dengan javascript. Contohnya: membuat/menghapus elemen html, event handling, menambahkan atribute, mengubah text, dsb.


Javascript & DOM #1 - DOM Tree

Setiap node pada DOM tree memiliki tipenya masing-masing, misalnya: element, attribute, text, dsb.

NodeList & HTML Collection:

  • Node List: merupakan kumpulan dari document node yang memiliki tipe berbeda- beda.
  • HTML Collection: merupakan kumpulan dari document node yang memiliki tipe yang sama. HTML collection ini bersifat live.

Struktur Hierarki DOM:

  • Root Node: merupakan node yang menjadi sumber dari semua node lain di dalam DOM, yang mana secara default node ini adalah document.
  • Parent Node: merupakan node yang berada 1 tingkat diatas node yang lain. Apabila sebuah node berada 2 tingkat diatas node lain maka node tersebut disebut grandparent, apabila 3 tingkat atau lebih maka disebut anchestor.
  • Child Node: merupakan node yang berada 1 tingkat dibawah node yang lain. Apabila terdapat beberapa node yang memiliki parent yang sama maka disebut siblings, sedangkan apabila sebuah node berada 2 tingkat atau lebih dari node lain maka node tersebut disebut dengan decendant.

DOM dan Javascript

  • DOM bukan fitur atau bahasa pemrograman, DOM merupakan representasi Object dari struktur halaman web.
  • DOM bukan bagian dari javascript, melainkan bagian dari WEB API.
  • DOM didesain secara agnostik, yang artinya dapat digunakan pada bahasa pemrograman apapun.

Document

Document merupakan representasi object dari halaman web. Dalam satu halaman web terdapat satu document yang sudah secara otomatis digenerate oleh browser ketika membaca file html. Document memiliki banyak sekali properties, contohnya: title, head, body, links, dsb.

Document Method

Document memiliki banyak method yang biasanya digunakan untuk emmanipulasi data DOM, misalnya membuat node, element, attribute, atau melakukan seleksi pada node yang sudah ada pada document.

Contoh seleksi node:

const tulisan = document.getElementById("tulisan");
console.log(tulisan);

Node

Node adalah base class dari docuemnt, element, dan attribute sehingga semua fitur yang dimiliki pada node dimiliki oleh turunannya. Node dalam DOM direpresentasikan sebagai Tree (Pohon DOM) sehingga terdapat beberapa hierarki seperti parrent, child, siblings, dsb.

Node Property

Node memiliki banyak sekali property. Diantara properti tersebut terdapat banyak propperty yang berfungsi sebagai reference ke node lainnya seperti reference ke node parent, children, atau siblingsnya.

Contoh:

// select node
const tulisan = document.getElementById("tulisan");

// accessing the properties
console.log(tulisan.textContent); console.log(tulisan.nextSibling); console.log(tulisan.previousSibling);

Node Method

Method pada node dapat digunakan untuk memanipulasi data node tersebut. misalnya menambah/menghapus Node children.


Element

Element merupakan node yang berbentuk element, yang biasanya merupakan elemen HTML seperti head, body, div, table, dsb.

Element Properties & Methods

Elemen juga merupakan turunan dari node sehingga semua fitur properties dan juga method dari node juga bisa digunakan pada elemen.

Contoh:

// select node
const elemen = document.getElementById("container");

// DOM manipulation
const node = document.createElement("p");
node.innerText = "lorem ipsum."
elemen.appendChild(node);
