Javascript & DOM #1

DOM adalah kepanjangan dari Document Object Model

DOM adalah antarmuka pemprograman untuk HTML yang mempresentasikan halaman web, sehingga program dapat mengubah dan memanipulasi strukturnya menurut Mozilla

Dalam DOM Tree, Window merupakan hierarki terluar sebelum document yang dimana.

Window dia bisa mengakses segala diluar dari web browser sedangkan document

Dia hanya berjalan di dalam browser

Javascript & DOM #2

Dalam DOM ada banyak tipe - tipe node yaitu :

  1. Element
  2. Atribut
  3. Text
  4. CData Section
  5. Entity Reference
  6. Entity
  7. Processing Instruction
  8. Comment
  9. Document
  10. Document Type
  11. Document Fragment
  12. Notation

Node List dan HTML Collection merupakan kumpulan node dan struktur array dari keduanya mirip

nodeList dapat berisi node apapun sedangkan HTML Collection harus berisi element HTML

HTMLCollection bersifat live sedangkan nodeList tidak.

Dalam struktur hierarki dom ada 3 yaitu :

  1. Root Node

    2. Node yang bersumber dari semua node lain di dalam node document

  2. Parent Node

    3. Node yang berada 1 tingkat diatas node yang lain

  3. Child Node

    4. Node yang berada 1 tingkat dibawah node yang lain

Tutorial Javascript DOM

JavaScript memiliki kemampuan untuk memanipulasi DOM, artinya dengan JavaScript, kita bisa mengakses tiap object di DOM bahkan memanipulasi object nya

DOM bukanlah bagian dari bahasa pemrograman JavaScript, melainkan bagian dari Web API

DOM memiliki representasi data

Document merupakan representasi object dari halaman web

Document secara otomatis dibuat dalam browser ketika membuka halaman web, sehingga kita tidak perlu membuat objek Document secara manual

Banyak method yang terdapat di document digunakan untuk memanipulasi data DOM, misal membuat Node, Element, Attribute atau mengambil dan menyeleksi Node di dalam document

Node adalah base class dari Document, Element dan Attr

Node dalam DOM itu bentuknya adalah Tree (pohon), artinya dengan Node, kita bisa melihat Parent (node diatas nya), Children (node dibawahnya), Sibling (node disebelahnya)

Element merupakan Node yang berbentuk element, biasanya adalah element HTML, misal html, head, body, div, table, dan lain-lain

Sebelumnya kita hanya mengambil element menggunakan id, artinya kita perlu buat dulu element nya di HTML Kita juga bisa membuat object element baru, dan menambahkannya ke halaman web

NodeList adalah kumpulan dari Node. Biasanya NodeList digunakan ketika kita menyeleksi banyak Node sekaligus, misal ketika kita ingin mengambil semua children di Node misalnya

NodeList memiliki 2 tipe, yaitu Live dan Static

HTML Element memiliki banyak sekali property dan method tambahan selain dari Element DOM
