SEO

Perarutaran dalam SEO adalah buat konten yang sangat baik, karena ketika jika konten yang buruk dan tidak menarik bagi manusia apalagi google sang mesin pencari. Dimana ketika konten yang bagus akan menarik seseorang untuk meklik halaman yang tampil di mesin pencarian. CTR, Click – Through Rate adalah seberapa banyak seseorang meklik link ketika tampil di SERP(Search Engine Ranking Page). CTR yang tinggi membuktikan bahwa konten relevan dengan judul dan deskripsi, dimana ketika klink link dan langsung Kembali ini di namakan “Bounce” ketika Bounce di lakukan secara cepat maka membuktikan konten tak relevan dan akan berdampak. Ketika tidak melakukan bounce, google akan mencatat Dwell Time yang adalah total waktu yang di habiskan sebelum Kembali.

Peraturan selanjut nya adalah Bot-Friendly HTML, dimana struktur HTML yang digunakan di website harus dapat dibaca oleh bot. Untuk dapat di pahami dengan bot Struktur HTML yang dapat digunakan adalah dengan menggunakan semantic HTML. Sematic HTML adalah dengan Konten utama harus masuk ke dalam tag "body",dapat diakses menggunakan tag "alt" dan gambar dan tag "aria", dan lain lain.

Peraturan terakhir adalah memuat HTML Anda yang telah dirender sepenuhnya dengan cepat. Terdapat 3 cara rendering Client side rendering dimana html di render di sisi client dengan kelemahan bot akan sulit megenalinya, lambat mengerti isi dari content. Static Generation atau pre render dimana seluruh page di build dan di cache di CDN kelemahan skala tidak baik dan data tidak update. Server side rendering dimana HTML di render di server dan tidak di render di client dimana ini memudahkan bot untuk membaca namun lambat dan data fetching ber ulang kali. Dari ketiga car ini terdapat cara lagi yaitu ISR, Incremental Static Regeneration dimana ini adalah format baru rendering di NEXT JS strategi ini akan merender halaman static ketika pertama kali website diakses lalu ketika ada request lain akan melakukan generate halaman static kembali pada background cara render ini cepat, baru, dan bot-friendly, namun tingkat deploy yang rumit, atau vendor lock in

SEO

Seluruh usaha yang bertujuan untuk menempatkan website di laman pertama hasil pencarian search enghine untuk kata kunci yang di inginkan disebut dengan SEO. SEO adalaha singkatan dari Search Engine Optimization. Adalah Teknik untuk mendapatkan pengunjung secara organic dan gratis dari mesin pencari. Hal teknis SEO sudah dintentukan oleh google :

  • Content, Konten dengan kualitas tinggi
  • Keyword, Menetapkan keyword secara tepat
  • Setup permalinks, title, description snippet, meta tag, inner-outer link pada konten
  • Optimisasi On Page
  • Optimisasi Off Page
