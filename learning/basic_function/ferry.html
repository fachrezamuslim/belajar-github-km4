FUNCTION pada JAVASCRIPT

#1 Intro

Menurut Douglas Crockford, function adalah kunci utama pada javascript yang membuat javascript sangat powerfull.

Menurut fungsi nya, function pada Javascript adalah:

  • Function adalah sub-program / sub-routine yang dapat dipanggil di bagian lain pada program.
  • Struktur dasar pembentuk dari JS Disebut juga sebagai prosedur atau kumpulan statement yang melakukan tugas / menghitung sebuah nilai
  • Harus dibuat terlebih dahulu agar dapat digunakan Termasuk ke dalam First-Class Object

Kenapa harus buat function:

  • Reusability (dapat digunakan berulang-ulang)
  • Dekomposisi (menyembunyikan kompleksitas program)
  • Modularitas

Kategori Function: Built-in function (fungsi yang sudah disediakan) User-defined function (fungsi yang dibuat user)

Cara buat function: Dengan deklarasi Dengan ekspresi

#2 Membuat dan Menjalankan FUNCTION

Function memiliki inputan lalu mengerjakan function tersebut dan menghasilkan output.

Function yang baik hanya mengerjakan satu tugas.

sintaks function:

function namaFunction(parameter/argumen) { // blok kode }

parameter adalah data yang dikirim dari luar saat pemanggilan function parameter dianggap sebagai variable khusus untuk function opsional atau boleh ada dan boleh tidak ada

Sebuah function mengembalikan nilai dengan keyword return

Setelah sampai return, function akan berhenti berjalan.

#3 Parameter dan Argument Function

Parameter adalah variable yang ditulis di dalam kurung function, parameter in digunakan untuk menampung nilai.

Argument adalah nilai yang dikirimkan ke parameter saat fungsi dipanggil

Jika parameter lebih sedikit dari argument, maka argument kelebihannya akan diabaikan.

Jika parameter lebih banyak dari argument, maka parameter kelebihannya akan diisi dengan nilai undefined.

arguments adalah array yang berisi nilai yang dikirimkan saat fungsi dipanggil.

#4 Refactoring Function

Refactoring adalah sebuah proses mengubah kode agar menjadi lebih baik tanpa mengubah fungsionalitas.

Kenapa harus melakukan refactoring:

  • mudah dibaca
  • mencegah duplikasi kode
  • mudah di uji
  • meningkatkan performa mudah di mantain

#5 Variable Scope Function

Scope membahas bagaimana sebuah variable dapat diakses dalam program.

Js menggunakan function scope.

#6 Function Declaration vs Function Epression

Sintaks function declaration

function identifier (parameter opsional ) {fumction body}

Sintaks function expression

function identifier opsional (parameter opsional) {function body}

keunggulan function declaration: lebih fleksibel mudah dipahami

keunggulan function epxression: lebih powerfull tapi harus didefinisikan terlebih dahulu
