Video 1 : FUNCTION pada JAVASCRIPT : Intro

Function merupakan kunci utama pada js, yang membuat js sangat powerfull. function adalah sebuah sub-program/ sub-routine yang dapat dipanggil dibagian lain program.

Manfaat menggunakan Function yaitu :

  • Reusability (DRY: Don't Repeat Yourself)
  • Dekomposisi / Abstraksi
  • Modularitas

Kategori function dibagi menjadi 2 yaitu :

Built-in Function

Contohnya yaitu : alert, confirm, prompt

User-defined Function

Contohnya yaitu : alert, confirm, prompt

Video 3 : FUNCTION pada JAVASCRIPT : PARAMETER dan ARGUMENT

Parameter adalah variabel yang ditulis didalam kurung pada saat function dibuat dan digunakan untuk menampung nilai yang dikirimkan saat function dipanggil. Argument adalah nilai yang dikirimkan ke parameter saat fungsi dipanggil.

variabel yang bernama arguments digunakan untuk menyimpan nilai array dari semua argument yang dikirim pada function. ketika jumlah parameter saat fungsi dibuat berbeda dengan jumlah argument yang dikirim yaitu:

  • Jika jumlah parameter saat fungsi dibuat lebih banyak dari jumlah argument yang dikirim saat memanggil fungsi maka parameter yang tidak diisi bernilai undefined
  • Jika jumlah parameter saat fungsi dibuat lebih sedikit dari jumlah argument yang dikirim saat memanggil fungsi maka argument tersebut tetap tersimpan pada fungsi

Video 4 : FUNCTION pada JAVASCRIPT : REFACTORING

Refactoring adalah sebuah proses mengubah kode agar lebih baik tanpa mengubah fungsionalitasnya. Adapun beberapa faktor mengapa refactoring harus dilakukan yaitu:

  • Readibility: kemudahan dalam membaca kode program
  • DRY (Don't Repeat Yourself): duplikasi kode
  • Testability: penulisan koda agar mudah saat proses pengujian
  • Performance: bagaimana cara menulis kode yang dapat meningkatkan performa dari program yang dibuat
  • Maintainability: bagaimada program dapat dengan mudah dikelola atau dikembangkan

Video 5 : FUNCTION pada JAVASCRIPT : VARIABLE SCOPE

Scope adalah bagaimana cara sebuah variabel dapat diakses dalam program. Dalam bahasa pemrograman terdapat dua jenis scope yaitu:

  • block scope: variabel yang dibuat didalam block scope tidak dapat digunakan diluar dari block-nya.
  • function scope: variabel yang dibuat didalam block scope dapat digunakan diluar dari block-nya.

ketika variabel dibuat tanpa dibungkus scope disebut global scope atau window scope. Global scope atau window scope dapat diakses pada block scope manapun. Deklarasi variabel didalam block scope atau saat menjadi parameter dengan nama yang sama dengan variabel global scope akan menjadi 2 variabel yang berbeda.

Video 6 : FUNCTION pada JAVASCRIPT : DECLARATION vs. EXPRESSION

Function Declaration

  • Lebih fleksibel (dapat ditulis dimanapun) karena pada javascript ada konsep Hoisting dimana deklarasi sebuah function pasti disimpan terlebih dahulu didalam memory
  • Mudah dipahami pemula

Function Expression

  • Harus didefinisikan terlebih dahulu sebelum dipanggil
  • Sebagai closure
  • Sebagai argumen untuk function lain
  • IIFE (Immediately Invoked Function Expression)
