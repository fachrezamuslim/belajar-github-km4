Belajar Function Video 1

Function adalah sebuah sub-program yang dapat 'dipanggil' dibagian lain dari program. untuk menggunakannya kita harus membuat terlebih dahulu functionnya lalu dipanggil. Termasuk dalam First-Class Object

Kenapa Menggunakan Function

  1. Reusability
  2. Dekompoposisi
  3. Modularitas

Belajar Function Video 2

Dalam function ada banyak kelebihannya yaitu :

  1. Blok kode yang dibuat untuk melakukan tugas spesifik
  2. Dapat Dipanggil Berulang Kali
  3. Memudahkan Penelusuran
  4. Reusability

Belajar Function Video 3

Parameter ialah variabel dalam kurung function yang digunakan untuk menampung nilai yang dikirim oleh function. Argument ialah parameter saat fungsi di panggil

Argument : array yang berisi nilai yang dikirimkan saat fungsi dipanggil

Belajar Function Video 4

Refactoring : sebuah proses mengubah kode agar menjadi lebih 'baik' tanpa mengubah fungsionalitasnya.

  1. Readbility :Kode yang dibuat untuk memudahkan dibaca oleh pembuat dan orang lain
  2. DRY ('Dont Reapeat Yourself') : untuk mempermudah penggunaan function agar lebih efisiensi
  3. Testability : Agar kode bisa mudah dalam pengujian
  4. Performance : Untuk memudahkan dalam meningkatkan performa dalam koding yang dibuat
  5. Mainbility : Agar program mudah dikelola agar bisa dikembangkan

Belajar Function Video 5

Scope : Lingkup variabel yang dimana dapat diakses Dalam program.

Dalam Scope ada 2 Tipe yaitu :

  1. Block Scope : Hanya berlaku dalam block yang dibuat saja tidak belaku untuk diluar dari block.
  2. Function Scope : Dapat digunakan di dalam blok yang dibuat ataupun block lain.

Belajar Function Video 6

Function Declaration

Jika ingin membuat function declaration maka : harus menulis code seperti ini :

function identifier (parameter opsional) {function body}

Sedangkan untuk membuat Function Expression harus menulis code seperti ini :

function identifier opsional (parameter opsional) {function body}


Kelebihan dari Function Declaration

  1. Lebih Fleksibel dapat ditulis dimanapun
  2. Mudah dipahami oleh pemula

Sedangkan untuk Function Expression yaitu :

  1. Harus didefinisikan terlebih dahulu sebelum dipanggil
  2. Lebih powerfull
    • Sebagai Closure
    • Sebagai Argument Function
    • IFFE (Immediately Invoked Function Expression)
