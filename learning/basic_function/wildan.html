FUNCTION pada JAVASCRIPT : Intro

Posted on March 30, 2023 by Wildan

Perkenalan Function

Funtion merupakan struktur dasar pembentuk dari javascript. Untuk dapat menggunakan function, kita harus membuat function terlebih dahulu untuk bisa digunakan dengan cara memanggil function tersebut. Function termasuk kedalam first class object, yang artinya function dapat disimpan dalam variabel, array, object, dan dapat juga dikirimkan sebagai parameter ke function lainnya.

Dengan menggunakan function kita dapat melakukan reusability atau DRY (Don't Repeat Yourself), serta melakukan dekomposisi dan modularitas yang artinya kita dapat memecah suatu program menjadi beberapa bagian yang lebih kecil. Dengan demikian, kita dapat memahami dan mengelola program yang kita buat dengan lebih mudah.

Pada javascript terdapat dua kategori function, yaitu function yang di buat oleh kita sebagai programmer (User-Defined Function) dan function yang di buat oleh javascript itu sendiri (Built-in-function).

Untuk membuat fungsi kita sendiri kita dapat menggunakan keyword function dan nama function (optional), lalu diikuti dengan tanda kurung yang berisi parameter (optional), dan diakhiri dengan tanda kurung kurawal yang berisi kode yang akan dijalankan ketika fungsi tersebut dipanggil. Function dapat mengembalikan nilai dengan cara menggunakan keyword return atau tidak.

Ada dua cara membuat function yaitu dengan cara function declaration dan function expression. Perbedaanya adalah function expression disimpan dalam variabel, sedangkan function declaration tidak.

Create and Run a Function

Function yang baik adalah function yang hanya mengerjakan satu hal saja.

Sandhika Galih memberikan contoh dalam membuat function jumlahVolumeDuaKubus dengan parameter a dan b. Function tersebut akan mengembalikan nilai dari hasil penjumlahan volume dari dua kubus yang memiliki sisi sepanjang a dan b.

Setelah sampai pada keyword return, interpreter akan langsung keluar dari function tersebut atau berhenti berjalan.

FUNCTION pada JAVASCRIPT : PARAMETER dan ARGUMENT

Posted on March 30, 2023 by Wildan

Perbedaan Parameter dan Argument ?

Parameter adalah variabel yang digunakan untuk menampung nilai yang di kirimkan ke function ketika function tersebut dipanggil. Sedangkan argument adalah nilai yang dikirimkan ke function ketika function tersebut dipanggil.

Pada Javascript jika parameter lebih sedikit dari argument, maka nilai yang tidak terpakai atau kelbihannya akan diabaikan. Namun jika parameter lebih banyak dari argument, maka nilai yang tidak terpakai akan diisi dengan nilai undefined.

Jika parameter lebih banyak dari argument, maka nilai kelebihannya akan diisi dengan nilai undefined.

Pseudo variabel arguments adalah sebuah variabel yang dapat digunakan di dalam function untuk mengakses semua argument yang dikirimkan ke function tersebut.

Refactoring

Refactoring merupakan sebuah proses untuk mengubah kode program yang sudah ada tanpa mengubah fungsionalitas dari program tersebut. Refactoring dilakukan agar code kita lebih readability atau mudah dibaca. DRY (Don't Repeat Yourself), Refactoring juga dilakukan agara code kita lebih reusable atau dapat digunakan kembali. Meningkatkan perfoamance dari code kita, Refactoring juga dilakukan agar maintanability dari code kita lebih mudah.

FUNCTION pada JAVASCRIPT : VARIABLE SCOPE

Posted on March 30, 2023 by Wildan

VARIABLE SCOPE ?

Dalam bahasa pemrograman terdapat dua jenis variable scope yaitu block scope dan function scope. Block scope adalah scope yang berada di dalam kurung kurawal, sedangkan function scope adalah scope yang berada di dalam function. Contoh block scope adalah bahasa pemrograman C. Javascript menganut function scope.

Jika kita membuat variable di dalam function, maka variable tersebut di anggap sebagai variable local. Variable local hanya dapat di akses di dalam function tersebut. Sedangkan jika kita membuat variable di luar function, maka variable tersebut di anggap sebagai variable global. Variable global dapat di akses di dalam function sedangkan variabel local tidak dapat di akses di luar function.

Sandhika Galih memberikan contoh apabila kita membuat variable di dalam function tanpa memberikan keyword var. Variable tersebut akan dianggap sebagai variable global.

FUNCTION pada JAVASCRIPT : DECLARATION vs. EXPRESSION

Posted on March 30, 2023 by Wildan

DECLARATION vs. EXPRESSION

Untuk penulisan function declaration kita harus menuliskan function identifier (parameterList opt) dan {function body}. Sedangkan untuk penulisan function expression kita harus menuliskan function identifier optional (parameterList opt) dan {function body}.

Function declaration lebih fleksibel (dapat ditulis dimanapun) karena konsep hoisting, dan mudah dipahami pemula. Sedangkan function expression harus didefinisikan terlebih dahulu sebelum digunakan. Lebih powerfull sebagai closure, sebagai argumen untuk function lain dan IIIFE (Immediately Invoked Function Expression).
