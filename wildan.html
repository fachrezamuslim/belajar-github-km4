Rangkuman Materi DOM

DOM singkatan dari document object model yaitu representasi element HTML pada dokumen menjadi sebuah object. DOM memiliki hirarki dimulai dari

  • root element
  • parent node
  • dan child node

Adapun beberapa tipe data dalam DOM adalah

  • Document
  • Node
  • Element
  • Node List
  • Attribute
  • NamedNodeMap

Untuk dapat mengakses document, kita bisa menggunakan object document.

Node adalah base class dari Document, Element, Attribute. Artinya semua fitur yang dimiliki di Node dimiliki diturunannya. Untuk memanipulasi Node kita bisa menggunakan Node Method, seperti menambah atau menghapus node children. Untuk membuat element kita bisa menggunakan method createELement(TAG)
